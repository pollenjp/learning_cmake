# Example 2: A Project with Directories

- https://github.com/derekmolloy/exploringBB/tree/master/extras/cmake/student

```
 ./
 |-- CMakeLists.txt
 |-- build
 |-- include/
 |   \-- Student.h
 \-- src/
     |-- Student.cpp
     \-- mainapp.cpp
```


## Execution

```
mkdir build
cd build
cmake ..
make
./testStudent
```
